## 📊 Featured Projects

### 🔍 [LLM Hallucination Calibration System](https://github.com/aaditbaldha2002/llm-uncertainty-calibration)
**Tech:** `FastAPI` • `FAISS` • `all-MiniLM-L6-v2` • `Llama 3` • `HuggingFace API` • `Chrome Extension` • `Python`  
Built a Chrome Extension + Python backend that detects potentially hallucinated LLM answers using a two-stage pipeline: sentence embeddings compared against a FAISS-indexed knowledge base (TruthfulQA, HotpotQA, SQuAD) for semantic similarity scoring, followed by Llama 3 claim verification via HuggingFace API. Returns a weighted confidence score (0–1), factuality verdict, and top supporting evidence passages.
